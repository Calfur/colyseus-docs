# Driver

## Overview

## Available Drivers

- [Local In-Memory (default)](#local-in-memory)
- [Redis](#redis)
- [MongoDB](#mongodb)

### Local In-Memory

The default driver used by Colyseus is the `LocalDriver`, which stores all room data in memory. This driver is suitable for development and small-scale applications.

### Redis

The `RedisDriver` stores room data in a Redis database. This driver is suitable for large-scale applications and is recommended for production environments.

```sh npm2yarn
npm install --save @colyseus/redis-driver
```
